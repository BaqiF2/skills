## 命令含义
design.md - 为项目制定规格说明，基于完整的 CLAUDE.md 项目文档
github/code-review.md - 对 pull request 进行代码审查
github/commit-push-pr.md - 提交、推送并创建 pull request
github/clean_gone.md - 清理所有标记为 [gone] 的 git 分支
spec_base/spec.md - 从 design.md 生成 spec.md 和 task.md
spec_base/task.md - 执行功能调整 task.md 的下一个任务
spec_tdd/spec.md - 从 design.md 生成 spec.md 和 task.md（TDD 版本）
spec_tdd/task.md - 执行 TDD task.md 的任务
