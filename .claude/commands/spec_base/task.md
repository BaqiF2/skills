## 核心任务流程
**必须按以下步骤执行任务：**

1. **读取理解**：仔细阅读$1的任务$2，明确任务要求和预期结果
2. **执行任务**：根据任务描述完成相应工作
3. **验证成功**：确认任务确实完成且符合要求
4. **标记完成**：将 **[ ]** 改为 **[x]**

⚠️ **重要**：只有验证成功后才可标记完成，验证失败必须重新分析解决

## 状态跟踪文件

当执行此命令时，会在 task.md 文件所在目录下自动生成一个 `task-xxx-status.md` 文件，用于跟踪任务执行状态。

### 文件命名规则
- 文件名格式: `task-{xxx}-status.md`
- `{xxx}` 来自任务编号，如果有多个编号，则使用_分隔
- 文件位置: 与执行的任务文件(task.md)相同的目录
