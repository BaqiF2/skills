本目录包含多组 Claude/插件相关的技能说明，供不同场景调用。

- verification-before-completion：完成前必须执行验证命令并提供证据。
- agent-sdk-dev：Claude Agent SDK 开发指南（工具、权限、MCP、会话等）。
- code-reading-guide：生成项目分层的代码阅读路径。
- claude/agent-development：Claude Code 插件 Agent 设计与规范。
- claude/command-development：Claude Code Slash Command 开发规范。
- claude/plugin-structure：插件目录结构与 `plugin.json` 配置规范。
- claude/skill-development：技能（SKILL）设计与组织规范。
- claude/mcp-integration：MCP 集成与配置指南。
- claude/plugin-settings：插件配置文件与 YAML frontmatter 读取规范。
- claude/hook-development：Hooks 事件驱动自动化与校验规范。
- claude/frontend-design/skills/frontend-design：高质量前端设计生成规范。
