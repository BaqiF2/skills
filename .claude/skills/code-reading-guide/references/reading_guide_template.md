2. `[核心Service文件2]` - YYY业务

**业务流程追踪**:

#### 场景1: 用户下单流程
```
1. OrderController.createOrder()
   ↓
2. OrderService.createOrder()
   - 校验库存
   - 创建订单
   - 扣减库存
   ↓
3. PaymentService.createPayment()
   - 生成支付单
   ↓
4. 返回订单详情
```

**关键决策点**:
- **库存扣减时机**: 下单时预扣,支付超时释放
- **事务边界**: OrderService.createOrder() 开启事务

**阅读成果检验**:
- [ ] 能描述3个核心业务流程
- [ ] 理解事务管理策略
- [ ] 知道异步处理场景
