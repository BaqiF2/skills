## 技能概览

### 1. Agent SDK 开发助手 (`agent-sdk-dev-zh`)

精通 Claude Agent SDK 开发的专家助手，提供以下方面的指导：

#### 核心能力
- **自定义工具和 MCP 服务器** - 使用 MCP（模型上下文协议）创建和集成自定义工具
- **安全与权限** - 设置权限控制、安全钩子和访问管理
- **子代理与并行处理** - 实现子代理用于并行任务执行和上下文隔离
- **会话与 I/O 管理** - 处理会话管理、流式输入/输出和结构化输出
- **代理技能与命令** - 配置 Agent Skills 和斜杠命令以增强功能
- **文件检查点与提示** - 自定义系统提示并实现文件检查点
- **成本跟踪与监控** - 跟踪 API 成本、监控使用情况并优化性能
- **部署与生产** - 安全托管智能体并部署到生产环境
